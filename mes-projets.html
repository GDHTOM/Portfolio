Java :

Création d'un programme java de tri automatisée de dépêche journalistique

Projet réalisé en binome

Résumé du projet

Ce projet d'étude consiste à créer un programme en Java capable d'analyser une dépêche journalistique et de déduire son sujet principal parmi quatre grandes catégories. Le programme utilise un lexique de mots avec des poids d'importance attribués à chaque mot. En comparant la dépêche aux mots du lexique, le programme effectue son analyse. Nous avons également rédigé un rapport en anglais de notre programme.

Compétences travaillées

Maitrise du language de programmation Java
Acquérir une expérience de travail d'équipe dans la programmation d'un programme.
Rédaction d'un rapport en anglais.
